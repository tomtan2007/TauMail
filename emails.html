One-time setup — about 5 minutes
This uses the Gmail API with your Google account directly — no third-party servers involved. You need a free Google Cloud project and an OAuth Client ID.
1
Go to console.cloud.google.com. Create a new project (top-left dropdown → New Project). Name it anything.
2
In the left sidebar go to APIs & Services → Library. Search for Gmail API and click Enable.
3
Go to APIs & Services → OAuth consent screen. Choose External, click Create. Fill in any App name and your email for support. Important: click Add or Remove Scopes and add https://www.googleapis.com/auth/gmail.readonly. Then under Test users, add your own email address. Save and continue through all steps.
4
Go to APIs & Services → Credentials → Create Credentials → OAuth client ID. Choose Web application. Under Authorized JavaScript origins add . Save and copy the Client ID.
The Client ID is stored only in your browser's localStorage. The app only requests read-only access to Gmail.

Sign in with Google to load your Canvas emails.
